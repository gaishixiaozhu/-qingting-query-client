# 蜻蜓智能志愿查询技能

## 简介

蜻蜓智能志愿查询是基于蜻蜓生涯数据库的智能志愿填报推荐系统，采用**等位分法**算法，为考生提供精准的志愿推荐服务。

## 功能特点

### 1. 智能意图识别

本技能能够智能识别用户查询意图：

| 意图类型 | 说明 | 触发示例 |
|---------|------|----------|
| 推荐志愿 | 生成完整志愿表 | "帮我推荐志愿"、"怎么填志愿" |
| 查专业 | 查询专业录取数据 | "计算机专业多少分"、"哪些学校有医学" |
| 查院校 | 查询院校历年数据 | "能上大连理工吗"、"东北大学录取分" |
| 简单问答 | 不触发技能 | "你好"、"你是谁" |

### 2. 三大核心功能

#### 🌟 志愿推荐
- 冲刺30% + 适合30% + 稳妥40% 比例分配
- 基于等位分法的精准风险评估
- 历年数据对比参考

#### 🔍 专业查询
- 按分数/位次筛选院校
- 专业录取数据查询
- 历年录取分数对比

#### 📊 院校查询
- 院校历年录取数据
- 分专业录取情况
- 录取概率评估

## 使用方法

### 标准查询格式

```
省份 + 科类 + 分数/位次 + [院校/专业/志愿]
```

#### 示例：

| 用户输入 | 系统识别 | 处理结果 |
|---------|---------|---------|
| 辽宁物理类520分能上什么学校 | 推荐志愿 | 生成志愿表 |
| 计算机专业哪些学校有 | 查专业 | 返回院校列表 |
| 大连理工录取分多少 | 查院校 | 返回历年数据 |
| 你好 | 简单问答 | 正常回答 |

### 风险等级说明

| 等级 | 分差范围 | 说明 |
|-----|---------|------|
| 🚀 冲刺 | 0~+20 | 有希望，需冲一冲 |
| ✅ 适合 | -10~0 | 把握较大，推荐 |
| 🛡️ 稳妥 | -20~-10 | 较稳，可保底 |
| 🔒 托底 | <-20 | 大概率录取 |

## 技术架构

```
用户输入 → 意图识别层(IRT) → 条件优化器 → 查询引擎 → 结果返回
                              ↓
                         等位分法计算
                              ↓
                         风险等级评估
```

### 等位分法算法

1. 考生分数 → 在一分一段表中 → 获取位次
2. 用该位次，在历年一分一段表中反查 → 获取历年等效分
3. diff_年 = 院校历年low_real - 考生历年等效分
4. avg_diff = mean(diff_2023, diff_2024, diff_2025)

## 注意事项

1. **三项必要信息**：省份、科类、分数/位次，缺一不可
2. **分数优先**：建议提供分数，系统自动转换位次
3. **历年数据**：系统优先使用2025年数据

## 错误处理

| 错误信息 | 说明 | 解决方法 |
|---------|------|---------|
| 缺少省份信息 | 未识别到省份 | 请明确说明省份 |
| 缺少科类信息 | 未识别到科类 | 请说明物理类/历史类 |
| 缺少分数/位次 | 未识别到分数 | 请提供分数或位次 |
| API Key无效 | Key配置错误 | 联系开发者 |
| 请求过于频繁 | 触发限流 | 等待后重试 |
